# StreamHub - Video Streaming Platform 🎬

A YouTube-like video streaming platform built with Flask for my college project.

## 🚀 Features
- Upload & stream MP4 videos
- View count tracking
- Dark mode YouTube-inspired UI
- SQLite database

## 🛠️ Tech Stack
Python, Flask, SQLite, HTML5, CSS3

## ⚙️ Run Locally
```bash
pip install flask
python app.py
